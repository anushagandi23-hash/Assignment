# Admin Credentials Reference

## Admin Login Credentials

These are the credentials for the admin account in the application. Only ONE admin account exists.

### Admin Account Details:
- **Username**: `admin`
- **Email**: `admin@modex.com`
- **Password**: `Admin@123`

### How to Login as Admin:

1. Go to the Login page (`/login`)
2. Check the **"Login as Admin"** checkbox
3. Enter either the username (`admin`) or email (`admin@modex.com`)
4. Enter the password (`Admin@123`)
5. Click **"Login as Admin"**
6. You will be redirected to the Admin Dashboard

### Admin Access:

Once logged in as admin, you can:
- View all shows/bus routes
- Create new shows
- Edit existing shows
- Delete shows
- Manage bookings
- View booking statistics
- Access the admin dashboard

### User Login:

Regular users can:
1. Sign up with a username, email, and password
2. Login with their username or email and password
3. Browse available bus tickets/shows
4. Make bookings
5. View their profile and user information

**Note**: User credentials are stored locally in the application (for demo purposes). In a production environment, these would be securely stored in a database.
